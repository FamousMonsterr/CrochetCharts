# Бабушкин Квадрат: Практическая Инструкция

Эта инструкция рассчитана на текущий интерфейс `Crochet Charts` из этого форка и использует реальные названия кнопок и горячих клавиш приложения.

## 1. Подготовка шаблона

Для бабушкиного квадрата удобнее всего работать так:

1. Создайте новый документ.
2. В панели `New Chart Options` можно сразу выбрать шаблон `Granny Square`.
3. Для этого шаблона теперь доступны отдельные параметры:
   - `Preset` — готовые варианты `4-Round Granny Square`, `6-Round Granny Square`, `Solid Granny Square`.
   - `Start` — `Magic Ring` или `Chain Ring`.
   - `Corner Arches` — число угловых арок в шаблонной геометрии.
4. Включите окно свойств: `P` или `Show Properties`.
5. Включите круглую сетку: `Round grid` или клавиша `G`, пока не появится круговая подложка.
6. В свойствах включите `Show Chart Center`, чтобы видеть центр мотива.
7. В выпадающем списке `guidelinesType` выберите `Rounds`.
8. Для круговой подложки задайте:
   - `Rounds` = число кругов мотива.
   - `Columns` = число опорных секторов.
   - `Cell Height` = расстояние между кругами.
9. Если вы хотите, чтобы петли автоматически поворачивались по радиусу, включите `Snap angle`.

Практически для бабушкиного квадрата обычно удобно сначала собрать один сектор или один угол, а потом размножить его поворотом, зеркалом или копированием.

## 2. Как выделять петли

В приложении есть три режима выделения:

1. `Box select` — прямоугольное выделение.
2. `Lasso select` — произвольное выделение.
3. `Line select` — выделение линией.

Переключение:

- кнопками на панели;
- клавишей `B` для циклического переключения режимов;
- через меню выбора режима.

Практика для бабушкиного квадрата:

- Для целого угла или стороны используйте `Box select`.
- Для сложного кластера из арок и столбиков лучше использовать `Lasso select`.
- Если надо быстро выбрать цепочку петель по диагонали или по дуге, полезен `Line select`.
- `Ctrl` при выделении добавляет объекты к уже выбранным.

## 3. Как двигать петли

Есть три основных способа перемещения.

### Мышью

1. Перейдите в режим `Stitch Edit` клавишей `D`.
2. Выделите одну или несколько петель.
3. Перетащите выделение мышью.

Если включена подложка, по завершении перемещения объект привязывается к текущей сетке.
Если включён `Snap angle` в режиме `Rounds`, петля не только встанет на радиус, но и автоматически повернётся по направлению к центру.

### Клавишами

В режиме `Stitch Edit`:

- `Left` / `Right` / `Up` / `Down` двигают выделение на 1 пиксель.

Это удобно для тонкой доводки углов бабушкиного квадрата.

### Через свойства

1. Выделите петлю или группу.
2. Откройте `Properties`.
3. Измените `X` и `Y`.

Этот способ удобен, когда нужно выставить одинаковые координаты для симметричных углов.

## 4. Как вращать петли

Есть три рабочих способа.

### Режим `Rotation Edit`

1. Нажмите `R`.
2. Выделите петлю или группу.
3. Используйте стрелки:
   - `Left` / `Up` = поворот на `-1°`
   - `Right` / `Down` = поворот на `+1°`

Это лучший способ для точной доводки наклона отдельных столбиков.

### Через свойства

1. Выделите объект.
2. В `Properties` измените поле `Angle`.

Это полезно, если нужен точный угол, например `45`, `90` или `135` градусов.

### Через окно `Copy, Mirror and Rotate`

1. Откройте окно `Show Mirror and Rotate` (`Shift+F8`).
2. Используйте кнопки:
   - `90`
   - `180`
   - `270`
   - `Custom`

Для бабушкиного квадрата это один из самых быстрых приёмов:

1. Нарисуйте один угол.
2. Выделите его.
3. Поверните на `90°`.
4. Повторите ещё 2 раза.

## 5. Как выравнивать петли

Откройте окно `Show Align and Distribute` (`Shift+F7`).

Доступны команды:

- `Align left sides`
- `Align right sides`
- `Align tops`
- `Align bottoms`
- `Align on a vertical axis`
- `Align on a horizontal axis`

Практика:

- Для одной стороны квадрата удобно сначала выровнять петли по верхнему или нижнему краю.
- Для четырёх углов мотива удобно выравнивать по вертикальной или горизонтальной оси.
- Если одна сторона квадрата «плывёт», выделите только элементы этой стороны и выровняйте их по одному краю.

## 6. Как равномерно разложить петли

В том же окне `Align and Distribute` есть блок `Distribute`.

Он умеет равномерно распределять выбранные элементы:

- по левым краям;
- по правым краям;
- по верхам;
- по низам;
- по центрам по вертикали;
- по центрам по горизонтали.

Для бабушкиного квадрата это полезно в двух местах:

1. Когда надо равномерно разложить группу столбиков по стороне.
2. Когда углы уже стоят правильно, но промежуточные петли на стороне расположены неравномерно.

## 7. Как быстро собирать симметричный мотив

Самый практичный workflow для бабушкиного квадрата:

1. Включите `Round grid`.
2. Включите `Show Chart Center`.
3. Нарисуйте только один угол или один сектор.
4. Выделите его `Box select`.
5. Если сектор состоит из нескольких элементов, при необходимости сгруппируйте: `Ctrl+G`.
6. Откройте `Show Mirror and Rotate`.
7. Используйте:
   - `Copy right` / `Copy left` / `Copy up` / `Copy down`, если хотите размножать соседние блоки.
   - `Rotate 90`, если хотите получить следующий угол.
   - `Mirror`, если нужна симметричная копия.
8. После копирования используйте `Align and Distribute`, чтобы точно собрать квадрат.
9. Если результат устраивает, при необходимости разгруппируйте: `Ctrl+Shift+G`.

## 8. Что лучше использовать именно для бабушкиного квадрата

Если вы рисуете мотив от центра наружу:

- Используйте `Round grid`.
- Включайте `Snap angle`.
- Стройте один угол, а не все четыре сразу.

Если вы правите уже почти готовый квадрат:

- Используйте `Box select`.
- Двигайте стрелками в `Stitch Edit`.
- Для точной геометрии применяйте `Align` и `Distribute`.

Если у вас боковые кластеры уже нарисованы, но углы «завалены»:

- Выделите угол.
- Поверните его в `Rotation Edit` стрелками по 1 градусу.
- Затем вручную добейте `X`/`Y` в `Properties`.

## 9. Полезные горячие клавиши

- `D` — `Stitch Edit`
- `R` — `Rotation Edit`
- `S` — `Scale Edit`
- `I` — `Indicator Edit`
- `W` — `Row Edit`
- `P` — показать свойства
- `B` — переключить режим выделения
- `G` — переключить тип подложки
- `Shift+F7` — `Align and Distribute`
- `Shift+F8` — `Mirror and Rotate`
- `Ctrl+G` — группировать
- `Ctrl+Shift+G` — разгруппировать
- `Delete` / `Backspace` — удалить выделение

## 10. Рекомендации по производительности на вашем Mac

Для старого MacBook Pro с `Intel HD 4000` и старой `NVIDIA GT 650M` лучше работать так:

1. В `Preferences -> Application` оставьте включённым `Low Graphics Load`.
2. Не держите одновременно много больших подложек-изображений.
3. Для крупных схем работайте по частям:
   - собрали угол;
   - выровняли;
   - только потом копируйте на остальные стороны.
4. Во время точной правки держите масштаб ближе к `100%`-`150%`, а не слишком высоким.
5. Если схема тяжёлая, временно отключайте лишние изображения и держите только нужный слой.

Этот форк уже включает облегчённый режим рендера и более дешёвую отрисовку изображений, чтобы снизить нагрузку на GPU на старых Mac.
